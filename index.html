Nnet port guard

SSH daemon remote ON/OFF controller against SSH attacks

View on GitHub Download .zip Download .tar.gz

What "nnet_port_guard" does?

If you have a remote server, probably SSH is open. Hackers know that and make SSH attacks to your server. There are some ways to protect like IPTABLES but if your SSH port is keep open, attacks will continue. Your CPU will busy to answer them.

What if SSH port is closed? No open port, no attack. This is why this program created.

If you install "nnet_port_guard" program to your remote server, you can Turn on or OFF SSH Daemon remotely. So you can protect against SSH attacks.
